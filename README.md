# Cálculo de Juros Simples em Python

Este projeto realiza o cálculo de juros simples com base no capital inicial, taxa de juros mensal e tempo em meses.

## Fórmula utilizada
J = C × i × t

Onde:
- C = Capital
- i = Taxa de juros
- t = Tempo

## Como funciona
O programa solicita ao usuário:
1. Valor do capital
2. Taxa de juros mensal (em %)
3. Tempo em meses

Ao final, exibe o valor total dos juros calculados.

## Tecnologias utilizadas
- Python 3

## Objetivo do projeto
Projeto desenvolvido com fins educacionais, visando praticar lógica de programação, entrada de dados e cálculos matemáticos.
